# DataScience_project_1 - Advanced EDA & Feature Engineering
Advanced Exploratory Data Analysis (EDA) and Feature Engineering on an E-commerce dataset using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Project Overview

This project focuses on cleaning and analyzing an e-commerce dataset using Python.

## Objectives

- Handle missing values
- Remove duplicate records
- Detect and treat outliers using IQR
- Perform Exploratory Data Analysis (EDA)
- Create new features for Machine Learning

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset

E-commerce Order Dataset

## Features Engineered

- OrderMonth
- OrderDay
- AverageItemPrice
- CouponUsed

## Output

- Cleaned Dataset
- Visualizations
- Correlation Analysis
